# Domain 2: Implement Image and Video Processing Solutions (15-20%)

Covers analyzing visual content using prebuilt and custom machine learning models on Azure.

## 🎯 Exam Objectives
- Analyze images using the Azure AI Vision service (OCR, spatial analysis, image tagging).
- Implement custom image classification and object detection with Custom Vision.
- Detect, analyze, and recognize faces using the Face service.
- Extract structured data from documents using Azure AI Document Intelligence.

## 📖 Key Study Topics

### 1. Azure AI Vision
*   **Image Analysis**: Generates tags, descriptions (captions), detects objects, and performs optical character recognition (OCR) via the Read API.
*   **Spatial Analysis**: Analyzes real-time video to monitor physical spaces (e.g., counting people in a store, detecting social distancing, or measuring dwell time).

### 2. Azure AI Custom Vision
*   **Classification**: Assigns one or more labels to an entire image (Multi-label vs. Multi-class).
*   **Object Detection**: Locates specific items in an image and provides their bounding box coordinates `[left, top, width, height]`.
*   **Training & Export**: Requires at least 15 images per tag. Models can be exported to run offline (ONNX, TensorFlow, CoreML, Docker container).

### 3. Face API
*   **Detection**: Locates human faces in an image, returning landmarks (eyes, nose, mouth) and attributes (age, emotion, head pose).
*   **Recognition**: Face Verification (1:1 match comparison) and Face Identification (1:N matching against a PersonGroup).
*   **Liveness Detection**: Prevents spoofing attacks by ensuring the submitted face is a live person rather than a photo/video.

### 4. Azure AI Document Intelligence (Form Recognizer)
*   **Prebuilt Models**: Invoices, receipts, ID documents, business cards.
*   **Layout Service**: Extracts text, tables, selection marks, and structure (headings).
*   **Custom Models**:
    *   *Template (neural/classification)*: For structured forms. Requires 5 documents.
    *   *Neural (generative)*: For unstructured documents (contracts, letters). Requires 5 documents.
