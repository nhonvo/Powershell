## ❓ Question 3

**Topic**: 

An organization is deploying applications to the cloud using platform as a service.

In the shared responsibility model, what remains the responsibility of the customer organization?

### Options:
- [ ] **A)** Operations
- [x] **B)** Data access policies [Correct]
- [ ] **C)** Physical security
- [ ] **D)** Network security

<details>
<summary><b>Reveal Answer & Explanation</b></summary>

**Correct Answer**: `B`

**Explanation**:
In the shared responsibility model, while the cloud provider manages the underlying infrastructure and platform security, the customer organization is responsible for data access policies, ensuring that only authorized users have access to their data and applications.
</details>
