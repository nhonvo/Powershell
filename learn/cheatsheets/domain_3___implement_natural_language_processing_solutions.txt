# Domain 3: Implement Natural Language Processing Solutions (30-35%)

Deals with text analysis, translation, language understanding, and conversational agent creation.

## 🎯 Exam Objectives
- Analyze text using Azure AI Language (sentiment, NER, PII detection, key phrase extraction).
- Translate text and documents using Azure AI Translator.
- Build custom conversational language models using Conversational Language Understanding (CLU).
- Design and implement custom question answering solutions.

## 📖 Key Study Topics

### 1. Azure AI Language
*   **Sentiment Analysis**: Returns positive, negative, neutral, or mixed sentiment scores at sentence and document levels. Includes opinion mining.
*   **Named Entity Recognition (NER)**: Identifies entities like locations, people, organizations, dates, and numbers.
*   **PII Detection**: Detects and redacts Personally Identifiable Information (SSNs, emails, phone numbers, credit cards).

### 2. Azure AI Translator
*   **Text Translation**: Translates text in real-time across over 100 languages. Supports auto-detection of source language.
*   **Document Translation**: Asynchronously translates entire complex layout documents (PDF, DOCX, XLSX) while retaining their original design and layout.
*   **Custom Translator**: Allows building customized translation models using industry-specific terminology.

### 3. Conversational Language Understanding (CLU)
*   **Intents**: Action/intent representing user goals (e.g., `BookFlight`, `CheckWeather`).
*   **Utterances**: Examples of what a user might say (e.g., "Book a flight to Paris tomorrow").
*   **Entities**: Variables extracted from utterances (e.g., `Destination = Paris`, `Date = tomorrow`).
*   **Deployment**: Train models and deploy them to endpoints for consumption by client applications or bots.
