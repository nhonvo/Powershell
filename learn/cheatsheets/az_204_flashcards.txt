# 🃏 AZ-204 Flashcards

## 🔵 Domain 1: Develop Azure Compute Solutions
Which file configures local settings and environment variables for Azure Functions?
?
**local.settings.json**.

---

## 🟢 Domain 2: Develop for Azure Storage
What Cosmos DB consistency level provides read-your-writes, monotonic read/write guarantees within a client session?
?
**Session Consistency**.

---

## 🟡 Domain 3: Implement Azure Security
What is the best configuration to allow App Services to pull secrets from Key Vault without storing local credentials in code?
?
**Managed Identity** combined with Key Vault access policies.
