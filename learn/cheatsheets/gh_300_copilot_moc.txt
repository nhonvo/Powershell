| Topic | Key Fact |
|:------|:---------|
| Public Code Filter threshold | **~150 characters** |
| Snippet retention — Business/Enterprise | **NEVER retained** |
| Snippet retention — Individual | **Retained by default** (must opt out) |
| Engagement data | **Always retained** (no opt-out) |
| PR Summaries | **Enterprise only** |
| Knowledge Bases | **Enterprise only** |
| Chat on GitHub.com | **Enterprise only** |
| Proxy cert fix | `NODE_EXTRA_CA_CERTS` → PEM file |
| Install Copilot CLI | `gh extension install github/gh-copilot` |
| Suggest terminal cmd | `?? <prompt>` |
| Explain terminal cmd | `what? <cmd>` |
| Generate tests | `/tests` chat command |
| Generate docs | `/doc` chat command |
| Explain code | `/explain` chat command |
| Exclusion file | `.github/copilot-exclusions.sha` |
| Policy cascade | **Enterprise → Org → Repo → User** |
